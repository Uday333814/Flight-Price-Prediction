#  Flight Price Prediction

This project predicts flight ticket prices using Machine Learning models.

##  Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost

##  Models
- Random Forest Regressor
- XGBoost Regressor
- Gradient Boosting

##  Performance
- Best Model: Random Forest
- R2 Score: ~0.81

## 📁 Files
- Notebook: Flight Price Prediction Project.ipynb
- Model: flight_model.pkl (not uploaded due to size)
